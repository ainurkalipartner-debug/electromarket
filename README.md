# ЭЛЕКТРОМАРКЕТ — сайт-каталог

React + Vite каталог электрооборудования. Цены не отображаются — все товары «под заказ» с запросом цены/КП через WhatsApp или email.

## Установка и запуск

```bash
npm install
npm run dev        # http://localhost:5173
npm run build       # production-сборка в /dist
npm run preview     # локальный просмотр production-сборки
```

## Обновление каталога из Excel

Номенклатура каталога (категории, товары, производители, фото) генерируется автоматически из прайс-листа поставщика:

```bash
npm run import:excel
```

Скрипт (`scripts/import-excel.mjs`) читает `ПРАЙС Торговый дом ИНВОЛЬТ 2026..xlsx`, распределяет позиции по категориям (правила — в `scripts/import/sheet-configs.mjs` и `section-router.mjs`), извлекает встроенные в Excel фото товаров и перезаписывает:

- `src/data/categories.js`, `src/data/manufacturers.js`, `src/data/products.js`
- `src/data/products/*.json` (товары по категориям + облегчённый поисковый индекс)
- `public/images/products/*`, `public/images/categories/*`
- `scripts/import-report.json` — счётчики по листам/категориям для самопроверки

Запускайте повторно при получении обновлённого прайс-листа от поставщика (с тем же именем файла в корне проекта).

## Известные ограничения

- **SEO для соцсетей**: сайт — клиентский SPA (без SSR). Заголовки/описания/OG-теги выставляются через `useSeo` на каждой странице и корректно видны Google, но боты предпросмотра ссылок (WhatsApp/Telegram/Facebook) не выполняют JS и увидят только статичные теги из `index.html`. Полноценное решение — пререндер/SSG, осознанно не делалось в этой итерации.
- **Формы без backend**: «Получить консультацию», «Запросить КП», «Отправить спецификацию», «Заказать звонок» формируют готовое сообщение и открывают WhatsApp/почтовый клиент — реальной отправки/хранения заявок на сервере нет.
- Категория «Кабель и провод» — пустая (в прайс-листе ИНВОЛЬТ нет номенклатуры силового кабеля).

## Деплой на Vercel

Проект статический (Vite SPA) — `vercel.json` уже настроен (SPA-rewrite + кэширование статики). Просто подключите репозиторий в Vercel, framework preset — Vite, build command `npm run build`, output directory `dist`.
